## 🚀 실전 시작 가이드

### Step 1: 환경 설정
```bash
cd solana-trading-bot

# API 키 설정
cat > .env <<EOF
BINANCE_API_KEY=your_api_key_here
BINANCE_API_SECRET=your_api_secret_here
EOF

# 의존성 패키지 설치
pip install -r requirements.txt
```

### Step 2: 테스트 모드 실행
```bash
# 테스트로 실행 (소액 테스트)
python3 simple_trading_bot.py

# 결과 확인
cat advanced_trading_results.json
```

### Step 3: 실전 모드 전환
```bash
# 소스코드 수정
# TEST_MODE 변수 제거 또는 False로 설정

# 실전 실행
nohup python3 simple_trading_bot.py > bot.log 2>&1 &

# 또는 tmux 사용
tmux new-session -s solana-trader
tmux send-keys -t solana-trader "cd solana-trading-bot && python3 simple_trading_bot.py"
```

### Step 4: 모니터링
```bash
# 로그 확인
tail -f bot.log

# 거래 상태 확인
curl -X GET http://localhost:8080/positions
```
